01 // LEARN — THE CAESAR CIPHER
THEORY

How It Works

Top: without cryptography a hacker intercepts and alters messages. Bottom: encryption scrambles the data — the hacker sees nothing useful.


Julius Caesar shifted each letter by 3 positions to protect military messages. The receiver knows the shift and reverses it.

Shift of 3 example:
A → D   B → E   C → F
X → A   Y → B   Z → C
Real example:
Message  : HELLO
Encrypted: KHOOR  (shift 3)

Each letter moves along the alphabet by a fixed number of steps.


Today this idea lives on in AES-256 (WhatsApp), TLS (HTTPS), and IEC 62443 industrial security standards.
